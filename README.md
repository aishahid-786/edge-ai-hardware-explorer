# Edge AI Hardware Explorer

Interactive edge AI deployment simulator modeling neural network optimization across NVIDIA Jetson and STM32 embedded systems. The platform visualizes real-time tradeoffs between latency, power consumption, accuracy, memory usage, and quantization strategies through dynamic hardware-aware performance estimation.

## Features

* Interactive neural network deployment explorer
* Jetson vs STM32 embedded hardware comparison
* FP32, FP16, INT8, and INT4 quantization simulation
* Real-time latency, power, accuracy, and memory estimation
* Dynamic deployment recommendation engine
* Embedded systems constraint visualization
* Responsive futuristic engineering UI
